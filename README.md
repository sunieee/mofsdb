# database

## 简介

此project用于：
1. 将现有多源数据库导入mysql
2. 使用sqlalchemy提供ORM接口，方便增删改查
3. 打包成pypi包，方便进行安装下载

现有数据库有：

- CSD
- QMOF
- CoRE

sqlalchemy教程：https://www.liaoxuefeng.com/wiki/1016959663602400/1017803857459008

mysql的连接：`engine = create_engine('mysql+mysqlconnector://root:digitalmofs@actvis.cn:9049/mofsdb')`
